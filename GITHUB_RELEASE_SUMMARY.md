## 🎯 立即開始

### 1. 克隆項目
```bash
git clone https://github.com/cis2042/ej-gemma-3n.git
cd ej-gemma-3n
```

### 2. 編譯運行
```bash
./gradlew assembleDebug
./gradlew installDebug
```

### 3. 獲取模型
```bash
cd scripts
./install_gemma_3n.sh
```
