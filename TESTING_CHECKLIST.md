# ✅ Android 應用測試檢查清單

## 🎯 測試目標
驗證 Gemma 3N 多模態 Android 助手的所有功能是否正常工作，並評估性能和用戶體驗。

## 📋 基礎環境測試

### 1. 編譯和安裝測試
- [ ] **項目克隆成功**
  ```bash
  git clone https://github.com/cis2042/ej-gemma-3n.git
  ```
- [ ] **編譯成功**
  ```bash
  ./gradlew clean assembleDebug
  ```
- [ ] **單元測試通過**
  ```bash
  ./gradlew test
  ```
- [ ] **APK 生成成功**
  - 檢查：`app/build/outputs/apk/debug/app-debug.apk`
- [ ] **安裝到設備成功**
  ```bash
  ./gradlew installDebug
  ```

### 2. 應用啟動測試
- [ ] **應用圖標顯示正常**
- [ ] **啟動畫面正常**
- [ ] **主界面加載成功**
- [ ] **無崩潰或 ANR**
- [ ] **啟動時間合理**（< 3秒）
